# தேர்வு தயார்ப்பு வலைத்தளம்

இந்த வலைத்தளம் TNPSC மற்றும் வங்கி தேர்வுகளுக்கான தயார்ப்பிற்காக வடிவமைக்கப்பட்டுள்ளது. தினசரி செய்திகள், தமிழ்நாடு புதுப்பிப்புகள் மற்றும் தமிழ் செய்தித்தாள்கள் ஆகியவற்றை உள்ளடக்கியது.

## வசதிகள்

### தேர்வு தயார்ப்பு
- **TNPSC தேர்வுகள்**: குரூப் 1, குரூப் 2 மற்றும் தமிழ் நிறுவன தேர்வுகளுக்கான தயார்ப்பு பொருள்
- **வங்கி தேர்வுகள்**: IBPS PO, IBPS Clerk மற்றும் SBI PO தேர்வுகளுக்கான தயார்ப்பு வழிகாட்டிகள்

### தினசரி புதுப்பிப்புகள்
- தினசரி செய்திகள்
- துப்புரவு செய்திகள்
- தமிழ்நாடு தினசரி புதுப்பிப்புகள்
- தமிழ்நாடு முக்கிய நிகழ்வுகள் மற்றும் பிறந்த நாள்கள்

### தமிழ் செய்தித்தாள்கள்
- தினமணி (Dinamani)
- இந்து தமிழ் (The Hindu Tamil)
- தீனகரன் (Dinakaran)
- மக்கள் குரல் (Makkal Kural)

## எப்படி பயன்படுத்துவது

1. இந்த கோப்புறையை வலை சேவையகத்தில் பொருத்தவும்
2. `index.html` கோப்பை உங்கள் வலை உலாவியில் திறக்கவும்
3. வழிசயல் பட்டைகளில் செய்திகளைப் பார்க்கவும்
4. "புதுப்பி" பொத்தான்களை தினசரி செய்திகளைப் புதுப்பிக்க பயன்படுத்தவும்

## தொழிநுட்ப விவரங்கள்

- **மொழி**: தமிழ்
- **எழுத்துரு**: Google Noto Sans Tamil
- **பதிலமைப்பு**: அனைத்து சாதனங்களுக்கும் பொருத்தமானது
- **திறன்கள்**: HTML5, CSS3, JavaScript ES6+

## தனிப்பட்டமயமாக்கல்

1. செய்திகளை புதுப்பிக்க `script.js` கோப்பில் உள்ள செயல்பாடுகளை மாற்றவும்
2. வடிவமைப்பை மாற்ற `styles.css` கோப்பைத் திருத்தவும்
3. புதிய பாடங்களைச் சேர்க்க பக்கத்தின் HTML ஐ மாற்றவும்

## குறிப்பு

இந்த வலைத்தளம் தேசிய, குறிப்பாக தமிழ்நாடு தேர்வு தேர்வர்களுக்கு உதவும் வகையில் வடிவமைக்கப்பட்டுள்ளது. தேர்வு வெற்றிக்கு நல்ல தயார்ப்பு ஆகுக!
